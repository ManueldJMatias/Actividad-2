Utilería.js

Mi librería de JavaScript para validar datos en formularios. Sin frameworks, solo JS puro.

Funciones que incluye

validarCorreo()

Checa si un correo tiene formato valido

validarCorreo("user@mail.com") // true

soloLetras()

Verifica que solo haya letras y acentos

soloLetras("María") // true

validarLongitud()

Valida cuantos digitos tiene un numero

validarLongitud(123, 5) // true

calcularEdad()

Calcula la edad desde la fecha de nacimiento

calcularEdad("2000-01-15") // 26

esMayorDeEdad()

Dice si la persona ya tiene 18

esMayorDeEdad("2005-03-10") // true

validarPassword()

Checa que tenga mayuscula, minuscula, numero, especial y 8+ caracteres

validarPassword("Abc123!@") // true

formatearTelefono()

Le da formato (XXX) XXX-XXXX a un telefono de 10 digitos

formatearTelefono("5551234567") // "(555) 123-4567"

contarPalabras()

Cuenta cuantas palabras tiene un texto

contarPalabras("Hola mundo") // 2
